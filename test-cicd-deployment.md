# CI/CD 部署测试

此文件用于测试完整的 CI/CD 流程。

## 测试内容

1. ✅ 本地提交到功能分支
2. ✅ 推送到 GitHub
3. ✅ 创建 Pull Request
4. ✅ 触发 CI/CD 检查（测试、代码检查、安全检查）
5. ✅ 自动合并到 main 分支
6. ✅ 云主机自动部署

## 预期结果

- 所有 CI/CD 检查应该通过
- PR 应该自动合并到 main
- 云主机应该收到部署通知并执行部署
- 部署成功后，可以通过 `/webhook/version` 查看版本信息
