# 统一管理后台开发计划

> **分支**: `feat/unified-admin-dashboard`
> **目标**: 创建统一管理界面，整合用户管理、留言管理、监控管理功能

## 📋 功能需求

### 1. 统一入口
- 单一登录页面
- 统一的导航栏和侧边栏
- 统一的身份认证和权限控制

### 2. 核心模块

#### 2.1 用户管理
- 用户列表展示
- 用户创建/编辑/删除
- 角色权限管理
- 密码重置

#### 2.2 留言管理
- 留言列表展示
- 留言查看
- 留言标记处理状态
- 留言删除

#### 2.3 监控管理
- 系统概览仪表板
- 实时监控数据
- 告警历史
- 健康检查

#### 2.4 知识库管理 (可选)
- 文章管理
- 分类管理
- 标签管理

#### 2.5 工单管理 (可选)
- 工单列表
- 工单详情
- 工单状态管理

### 3. 技术设计

#### 3.1 路由结构
```
/admin/
  /                    - 仪表板首页
  /login              - 登录页面
  /logout             - 登出
  /users/             - 用户管理
  /messages/          - 留言管理
  /monitoring/        - 监控管理
  /kb/                - 知识库管理 (可选)
  /case/              - 工单管理 (可选)
```

#### 3.2 权限控制
- 基于角色的访问控制 (RBAC)
- Admin: 全部权限
- Customer Service: 工单、留言管理
- Viewer: 只读权限

#### 3.3 UI 设计
- 响应式布局 (Bootstrap/Tailwind)
- 暗色主题支持
- 实时数据刷新
