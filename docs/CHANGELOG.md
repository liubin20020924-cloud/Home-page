## [2.0.0] - 2026-03-07

### 新增 (Added)

#### 留言系统增强功能
- 联系表单功能 (`feat/contact-form-features`)
  - 咨询类型选择（开通账户/技术咨询）
  - 联系电话必填字段
  - 开通账户自动创建用户账户
  - 技术咨询仅发送邮件通知
- 留言管理功能
  - 留言列表展示（支持分页和状态筛选）
  - 留言详情查看
  - 留言状态更新（pending/processed/completed）
  - 批量删除留言
  - 留言搜索功能
- 留言回复功能
  - 管理员在线回复留言
  - 回复内容保存到数据库
  - 回复后自动发送邮件通知客户
  - 回复历史记录
  - 回复状态跟踪（draft/sent/failed）
  - 回复模板快速应用
- 账户激活功能
  - 管理员激活账户
  - 自动生成临时密码
  - 邮件发送账户信息（用户名、密码、登录地址）
  - 首次登录强制修改密码提示
  - 激活时间记录
- 回复模板管理
  - 创建、编辑、删除回复模板
  - 模板分类管理（通用/账户/技术/计费/其他）
  - 模板变量替换（{name}, {email}, {phone}, {company_name}, {message}, {username}）
  - 模板使用统计
  - 系统预设模板（5个）
  - 模板启用/禁用控制

#### 数据库脚本
- 智能数据库同步脚本 (`sync_database.py`)
  - 自动检测数据库状态
  - 只补充缺失的内容
  - 不影响现有数据
  - 安全、可重复执行
  - 详细的执行日志
  - Windows/Linux/macOS 支持
- 数据库升级脚本 (`upgrade_to_v2.sql`)
  - 升级现有数据库结构
  - 添加留言回复相关字段
  - 添加咨询类型字段
  - 添加注册来源和激活时间字段
  - 创建回复模板表
- 完整初始化脚本 (`init_database.sql` v2.0)
  - 整合所有补丁脚本内容
  - 包含所有数据库、表和字段
  - 包含所有索引
  - 包含系统预设数据
  - 无需执行补丁脚本

#### 邮件服务增强
- 留言提交通知邮件
  - 发送到 support@cloud-doors.com
  - 包含留言人信息、公司、联系方式
  - 根据咨询类型区分内容
- 留言回复通知邮件
  - HTML 格式美化
  - 包含原始留言和回复内容
  - 发送失败重试机制（2次）
  - 超时时间 60 秒
- 账户激活邮件
  - 包含用户名和临时密码
  - 包含登录地址
  - 首次登录强制修改密码提示
  - 联系方式信息
- 企业微信邮箱支持
  - SMTP_SSL 连接（端口 465）
  - 支持域名和 IP 地址
  - 本地 hostname 设置

## 版本说明

### 版本号格式

```
主版本号.次版本号.修订号 (MAJOR.MINOR.PATCH)
```

- **主版本号 (MAJOR)**: 不兼容的 API 修改
- **次版本号 (MINOR)**: 向下兼容的功能性新增
- **修订号 (PATCH)**: 向下兼容的问题修正

### 变更类型

- **新增 (Added)**: 新功能
- **变更 (Changed)**: 功能变更
- **废弃 (Deprecated)**: 即将移除的功能
- **移除 (Removed)**: 已移除的功能
- **修复 (Fixed)**: Bug 修复
- **安全 (Security)**: 安全漏洞修复
